NASMのプリプロセッサについてちょっとだけ調べました。

<!-- more -->

## 単純な置換
```
%define count 42
```
countを42に置換する

## マクロ
```
%macro test 3
dq %1
dq %2
dq %3
%endmacro
```
`test3`という文字列を展開する。
